Estrutura Básica do Modal

Diagrama da estrutura do modal Bootstrap

3 Formas de Acionar um Modal

1. Via Atributos HTML

A forma mais simples, usando apenas data-attributes no HTML.

2. Via JavaScript Puro

Controle total através do JavaScript.

3. Via Evento de Página

Modal que abre automaticamente após 2 segundos.

O modal abrirá automaticamente...
Modal Acionado por HTML

Este modal foi aberto usando apenas atributos HTML:

<button data-bs-toggle="modal" data-bs-target="#modalExemplo1">
Modal Acionado por JavaScript

Este modal foi aberto através de JavaScript puro:

const modal = new bootstrap.Modal('#modalExemplo2');modal.show();
Modal Acionado por Evento

Este modal abriu automaticamente após 2 segundos usando:

setTimeout(() => {const modal = new bootstrap.Modal('#modalExemplo3'); modal.show();}, 2000);
