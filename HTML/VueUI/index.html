Grid Responsivo

O Grid do Bootstrap é um sistema para criar layouts flexíveis. Ele divide a tela em 12 colunas. Dizemos aos elementos quantas dessas 12 colunas eles devem ocupar em diferentes tamanhos de tela. Isso permite que o site se adapte perfeitamente a celulares, tablets e desktops.

Lista de Cards com v-for

A diretiva v-for do Vue.js é um "laço de repetição" para o HTML. Ela pega um array de dados do nosso JavaScript (no caso, a lista de produtos) e repete um bloco de HTML para cada item do array. Isso é extremamente útil para criar listas, tabelas e galerias de forma dinâmica, sem precisar escrever o código para cada item manualmente.

Modal Bootstrap

Um Modal é uma janela "pop-up" que aparece sobre o conteúdo principal. O Bootstrap torna sua criação muito fácil usando atributos data-bs-*. O botão usa data-bs-target="#exemploModal" para "mirar" no modal que tem o id="exemploModal". Toda a lógica de abrir e fechar é controlada pelo JavaScript do próprio Bootstrap.

v-if vs v-show

Ambas as diretivas escondem ou mostram elementos, mas funcionam de forma diferente. v-if literalmente remove o elemento do código HTML quando a condição é falsa, destruindo-o. Já v-show apenas aplica o estilo CSS display: none, mantendo o elemento no código, apenas invisível.

v-model

A diretiva v-model é uma das mais poderosas do Vue. Ela cria uma "ligação de dados de mão dupla" (two-way data binding). Isso significa que, se você digitar no campo, a variável no JavaScript muda. E se a variável no JavaScript mudar por outro motivo, o campo no HTML será atualizado automaticamente.
