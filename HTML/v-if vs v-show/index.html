V-IF vs V-SHOW

v-if: Adiciona ou remove completamente o elemento do DOM com base na condição.

Isso significa que, se a condição for falsa, o elemento não estará presente no DOM, bom para situações em que o elemento nao precisa ser renderizado ate que a condição seja verdadeira, econimizando recursos

v-show: Alterna a visibilidade usando CSS display: none, mas o elemento permanece no DOM o tempo todo.
Isso faz com que o elemento esteja sempre no DOM, bom quando voce precisa alternar a visibilidade do elementos muitas vezes sem a necessidade de recriação/destruição do DOM.
